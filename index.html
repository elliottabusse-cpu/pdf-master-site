⚠️ Premier lancement — avertissement macOS

  1. Double-cliquez sur PDF.Master.pkg pour lancer l'installation.
  2. Si macOS affiche un avertissement de sécurité, cliquez Terminé (ne pas placer dans la corbeille).
  3. Allez dans Réglages Système → Confidentialité et sécurité, faites défiler vers le bas et cliquez "Ouvrir quand même".
  4. L'app s'installe dans /Applications et s'ouvre normalement. Cette étape n'est requise qu'une seule fois.

⚠️ Premier lancement — avertissement Windows

  1. Double-cliquez sur PDF.Master.Setup.exe pour lancer l'installation.
  2. Si Windows affiche "Windows a protégé votre PC", cliquez Informations complémentaires.
  3. Cliquez ensuite "Exécuter quand même".
  4. L'app s'installe et se lance automatiquement. Cette étape n'est requise qu'une seule fois.
